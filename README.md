# Market Analysis for Digital Fitness Products

## 1) Data Background
As a product manager for a fitness studio the goal is to understand the current demand for digital fitness classes
We need to gauge if there is enough market interest to justify creating new digital products
To figure this out we analyze global Google Trends and YouTube search data
The datasets track search volumes for keywords like workout home workout gym workout and home gym over several years

## 2) Analysis Method
First we loaded the global search interest data to see how the popularity of fitness keywords changed over time
We converted the weekly search data into a monthly average to identify peak periods of interest
Then we compared different keywords to see if people were more interested in working out at home or at the gym
Finally we analyzed geographic data to find which specific countries had the highest demand for these fitness trends

## 3) Libraries Used
We used pandas to import the CSV files and manipulate the time series data
Pandas made it easy to group the data by month and filter for the top countries
We used matplotlib and seaborn to create visual plots of the search trends over time
These visualizations helped us clearly see the spikes in interest and compare regional demand

## 4) Results & Conclusion
The data showed a massive spike in fitness searches in April 2020 right when global lockdowns started
During this time search interest shifted heavily from gym workout to home workout and home gym
The United States and a few other key countries consistently ranked at the top for fitness search interest
Even after the pandemic peak there is still a strong sustained market for at-home fitness content
The company should move forward with developing digital fitness classes and target these high-interest regions
